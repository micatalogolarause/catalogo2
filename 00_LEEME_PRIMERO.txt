════════════════════════════════════════════════════════════════════════

Para acceder desde otra computadora:

1. Obtén tu IP local:
   cmd → ipconfig

2. Accede desde:
   http://TU_IP/catalogo2

Para acceso por Internet (IP pública):
├── Configura router/firewall
├── Abre puertos 80 (HTTP) y 443 (HTTPS)
├── Usa dominio o DNS dinámico
└── Implementa SSL/TLS

🆘 SOLUCIÓN DE PROBLEMAS
════════════════════════════════════════════════════════════════════════

Problema:              Solución:
─────────────────────────────────────────────────────────────────────
No se conecta BD    → Verifica MySQL en XAMPP
URLs no funcionan   → Habilita mod_rewrite en Apache
Imágenes no se ven  → Verifica carpeta public/images
Login no funciona   → Limpia cookies del navegador
Error 404           → Verifica que Apache esté activo
